一、拦截垃圾评论SPAM

据说来源自anti-spam，出自某大神之手【（惊恐）绝壁不是卤煮】：

/* 垃圾评论拦截（代码流传已久，请注意搜寻迭代更新版本）*/  
class anti_spam {  
   function anti_spam() {  
       if ( !current_user_can('level_0') ) {  
         add_action('template_redirect', array($this, 'w_tb'), 1);  
         add_action('init', array($this, 'gate'), 1);  
         add_action('preprocess_comment', array($this, 'sink'), 1);  
       }  
     }  
   function w_tb() {  
   if ( is_singular() ) {  
     ob_start(create_function('$input','return preg_replace("#textarea(.*?)name=([\"\'])comment([\"\'])(.+)/textarea>#",  
     "textarea$1name=$2w$3$4/textarea><textarea name=\"comment\" cols=\"100%\" rows=\"4\" style=\"display:none\"></textarea>",$input);') );  
   }  
}  
function gate() {  
   if ( !emptyempty($_POST['w']) && emptyempty($_POST['comment']) ) {  
     $_POST['comment'] = $_POST['w'];  
   } else {  
     $request = $_SERVER['REQUEST_URI'];  
     $referer = isset($_SERVER['HTTP_REFERER'])         ? $_SERVER['HTTP_REFERER']         : '隐瞒';  
     $IP      = isset($_SERVER["HTTP_X_FORWARDED_FOR"]) ? $_SERVER["HTTP_X_FORWARDED_FOR"] . ' (透过代理)' : $_SERVER["REMOTE_ADDR"];  
     $way     = isset($_POST['w'])                      ? '手动操作'                       : '未经评论表格';  
     $spamcom = isset($_POST['comment'])                ? $_POST['comment']                : null;  
     $_POST['spam_confirmed'] = "请求: ". $request. "\n来路: ". $referer. "\nIP: ". $IP. "\n方式: ". $way. "\n內容: ". $spamcom. "\n -- 记录成功 --";  
   }  
}  
function sink( $comment ) {  
   if ( !emptyempty($_POST['spam_confirmed']) ) {  
     if ( in_array( $comment['comment_type'], array('pingback', 'trackback') ) ) return $comment;  
     /*方法一: 直接挡掉, 將 die(); 前面两斜线刪除即可.*/

二、拦截非中文开头的评论

（大多数垃圾评论都是用网址、日文等开头的，碎碎念只拦截小日本和大米的评论）：

/* 禁止全英文和日文评论 */  
function ssn_comment_post( $incoming_comment ) {   
   $pattern = '/[一-龥]/u';   
   $jpattern ='/[ぁ-ん]+|[ァ-ヴ]+/u';   
   if(!preg_match($pattern, $incoming_comment['comment_content'])) {   
       err( "写点汉字吧，博主看不懂鸟语！ Please write in chinese ！" );   
       }   
       if(preg_match($jpattern, $incoming_comment['comment_content'])){   
           err( "日文滚粗！Stop！" );   
           }   
           return( $incoming_comment );   
           }   
add_filter('preprocess_comment', 'ssn_comment_post');

第二段这个拦截代码很实用，比如在线投稿的审核就可以直接在投稿输入窗口拦截掉非法投稿、垃圾投稿、恶意投稿。

灵活运用第二段代码，你会发现，原来小小的一段代码可以千变万化出各种功能；
