CENTOS7下安装配置nodejs（wget方法）

2017-07-14

在开始本文之前，默认您已参考前文《CENTOS7下从零开始配置nginx并开机启动》安装并配置必须的编译环境。

参考教程：Centos7下配置node.js环境

下载并安装nodejs

# 编译环境
yum -y install gcc gcc-c++ openssl-devel
# 下载并安装
wget http://nodejs.org/dist/v8.0.0/node-v8.0.0.tar.gz
tar zxvf node-v8.0.0.tar.gz
cd node-v8.0.0
./configure --prefix=/usr/local/node
make && make install

配置node环境

vim /etc/profile

在文本的最后添加一下内容：

export NODE_HOME=/usr/local/node
export PATH=$NODE_HOME/bin:$PATH
export NODE_PATH=$NODE_HOME/lib/node_modules:$PATH

保存退出。

source /etc/profile

检验node

node -v

开机自动运行nodejs脚本

/etc/profile.d/下新建脚本customer.sh:

nohup node /var/public_root/www/app.js &

关闭node服务

# 检查node进程
ps -aux|grep node
# 关闭进程,xxxx 为进程对应pid
kill -9 xxxx
